# NAME

App::mdweb - Markdown server and file browser

# SYNOPSIS

    $ mdweb daemon
    $ mdweb daemon --listen http://127.0.0.1:5000
    $ mdweb daemon --listen http://*:8080

# DESCRIPTION

This webserver can act as a file browser and markdown renderer.
